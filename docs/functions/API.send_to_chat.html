Function send_to_chat

  • send_to_chat(escort: IDataEscort): Promise<void>

  • Событие для отправки сообщения в лобби.
    Используемый пакет:

    {
     chat_id: string
     message: string
    }

    В случае успеха создает одноименный ивент и отправляет на него JSON объект:

    {
     complete: true
    }

    Все события из чата матча будут приходить на event chat в формате:

    {
     "chat_id":"lobby#xxxx",
     "message":
     {
       "from": "system or username",
       "message": "text of message"
     }
    }

    • escort: IDataEscort
